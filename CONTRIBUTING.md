# Contributing

Thank you for your interest in contributing to optics-spec-checker.

## How to Contribute

You can contribute by:

- Reporting bugs
- Suggesting new optical parameters
- Improving documentation
- Adding tests
- Refactoring code for readability

## Development Guide

1. Fork this repository.
2. Create a new branch.
3. Make your changes.
4. Run the project and tests.
5. Open a pull request.

## Project Goals

This project aims to provide a simple and educational tool for checking optical lens design specifications.

Contributions that make the tool easier to understand, use, or extend are welcome.
